Weather Data Retrieval (using Python + API + Pandas)

A Python REST API app that retrieves weather data from WeatherAPI.com using requests module and analyzes results using pandas

Features:
- Retrieves current weather data from a user-inputted city
- Retrieves the last 7 days of historical temperatures (max/min/avg) and returns a pandas dataframe
- Indetifies the hottest and coldest day from the last 7 days using min/max temps

Requires:
Request Module
Pandas Module
Python

Run this program using terminal/powershell/IDE of your choice.
